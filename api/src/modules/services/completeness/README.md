# Completeness Engine

Completeness Engine bertanggung jawab untuk **menghitung dan menyimpan status kelengkapan dokumen pegawai** terhadap suatu layanan.

Engine ini digunakan oleh:

- Services Engine
- Workflow Engine
- Admin tools
- Batch job rekalkulasi kelengkapan
